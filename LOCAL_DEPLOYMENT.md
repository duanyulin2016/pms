# 护照管理系统：本地部署指南

本应用已优化为**全栈架构**，支持完全脱离互联网的本地部署。

## 核心优势
1.  **数据安全**：数据存储在本地 `data/db.json` 文件中，不上传云端。
2.  **局域网访问**：同一网络下的用户可通过 IP 地址共同使用。
3.  **零依赖**：不需要注册 Firebase 或连接外网。

## 本地部署步骤

### 1. 运行环境准备
*   安装 [Node.js](https://nodejs.org/) (建议版本 v18+)。
*   将项目代码拷贝到目标电脑。

### 2. 初始化与运行
在项目根目录下打开终端并执行：
```bash
# 安装依赖
npm install

# 启动服务
npm run dev
```

### 3. 如何在其他电脑上访问
1.  找到运行服务器电脑的**局域网 IP 地址**（例如 `192.168.1.5`）。
2.  其他电脑在浏览器输入：`http://192.168.1.5:3000`。

### 4. 数据备份
*   所有数据（用户信息、护照记录、日志）都保存在项目目录下的 `data/db.json` 文件中。
*   建议定期复制该文件以备份数据。
